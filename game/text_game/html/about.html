Prologue

夢を見ていた気がする。
とても、とても大切だった誰かの笑顔。声。温もり。綺麗な黒い髪。
その髪に光る白いヘアピンはボクが誕生日に贈ったモノだ。
とても暖かくて、かけがいのないモノだったはずなのに、
それらはボクの体を通り抜け、どこか遠くへ行ってしまう気がして、
ひどく不安な気持ちになる…
